# EoSII

This project is focused on fixing the original EoSII codebase we based ours off of.

  - It now compiles on modern systems under both gcc and clang.
  - Merged back in various QOL features from derivative MUD Dragon's Exodus
  - Kept it mostly stock unless a change was required to fix an underlying issue.
  - I did not port over any combat or major engine changes, combat should feel exactly as it did in the initial release with the exception of damage numbers.

No support is guaranteed for this release and all existing licensing agreements remain in place. This project was purely for fun and out of respect for the MUD that started it all for us. Please feel free to fork and start your own game based off of it and if you want to chat MUD code you can always find me on Dragon's Exodus!
